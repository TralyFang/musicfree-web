# MusicFree Web 第二阶段 - 补充优化方案

基于原生 APP（MusicFree v0.6）的功能和 UI 对比分析，列出 Web 端缺失的功能、手机端 UI 适配问题和交互优化点。

---

## 一、功能缺失清单

### 1.1 搜索功能增强

| 优先级 | 缺失功能 | APP 实现 | 说明 |
|--------|---------|---------|------|
| P0 | 搜索分类标签 | TabView: 音乐/专辑/歌手/歌单 | Web 端目前只搜索音乐类型，APP 支持按类型切换 |
| P0 | 搜索历史 | historyPanel + localStorage | 记录搜索历史，点击可快速搜索，支持清除 |
| P1 | 搜索加载更多 | 分页滚动加载 | Web 端目前只加载第一页，无翻页/滚动加载 |
| P1 | 歌单内搜索 | searchMusicList 页面 | 在歌单/播放列表内按关键词筛选歌曲 |
| P2 | 搜索建议/热词 | 插件提供 | 输入时展示搜索建议 |

### 1.2 播放详情页增强

| P1 | 定时关闭 | setScheduleCloseTimeDialog | 设置定时停止播放 |

| P2 | 封面旋转动画 | 圆形封面 + 旋转动效 | APP 播放时封面有旋转动画 |

## 二、手机端 UI 适配问题

### 2.1 布局问题

| 问题 | 当前状态 | 优化方案 |
|------|---------|---------|
| 底部播放栏遮挡内容 | `pb-24` 固定 padding | 动态计算，确保最后一项不被遮挡 |
| 移动端侧边栏过宽 | 固定 `w-60` | 移动端改为全屏或 80% 宽度抽屉 |
| 首页快捷入口太小 | 卡片式布局 | 参照 APP 改为图标+文字的大按钮网格（推荐歌单、榜单、播放历史、本地音乐） |
| 搜索输入框不突出 | 普通 input 样式 | 移动端顶部搜索栏固定，参照 APP 的圆角搜索框 |
| 播放详情页非全屏 | 普通路由页面 | 移动端应为全屏覆盖层（从底部滑入），桌面端可保持侧栏布局 |

### 2.2 触控交互问题

| 问题 | 当前状态 | 优化方案 |
|------|---------|---------|
| 歌曲项无长按操作 | 仅点击播放 | 添加长按（touch hold）触发操作面板 |
| 进度条拖拽不友好 | 仅点击定位 | 添加 touch 拖拽支持，放大触控区域 |
| 无手势操作 | 无 | 播放详情页支持左右滑动切换封面/歌词 |
| MusicBar 无点击跳转 | 无 | 点击 MusicBar 歌曲信息区域跳转到播放详情页 |
| 列表无下拉刷新 | 无 | 推荐/排行榜等页面支持下拉刷新 |
| 无回到顶部 | 无 | 长列表提供回到顶部按钮 |

### 2.3 视觉体验问题

| 问题 | 当前状态 | 优化方案 |
|------|---------|---------|
| 无加载骨架屏 | 白屏等待 | 添加 Skeleton 加载占位 |
| 无空状态插图 | 纯文字 "暂无数据" | 添加 empty 插图组件 |
| 歌曲封面无占位 | 直接显示 logo.png | 添加渐变加载动画 |
| 无 Toast 提示 | 无反馈 | 操作成功/失败时显示轻量 Toast |
| 播放页背景单调 | 纯色背景 | 使用封面图做高斯模糊背景（参照 APP） |
| 无页面过渡动画 | 硬切换 | 添加路由过渡动画（淡入/滑动） |

## 五、技术实现要点

### 全屏播放详情（移动端）

```
方案：使用 React Portal + CSS transform
- 从底部滑入：transform: translateY(0) / translateY(100%)
- 下滑关闭：监听 touchmove Y 轴位移，超过阈值关闭
- 桌面端：保持当前路由式布局
```

### 操作面板

```
方案：统一 Panel 组件
- 移动端：BottomSheet（从底部弹出）
- 桌面端：ContextMenu（右键菜单）
- 通过 usePanel hook 统一调用
```

### 搜索分类

```
方案：Tab 组件 + 懒加载
- 每个 Tab 独立状态（分页、加载状态）
- 切换 Tab 不重新请求已有数据
- 支持 ICommon.SupportMediaType: 'music' | 'album' | 'artist' | 'sheet'
```

### 虚拟列表

```
方案：自实现或引入 @tanstack/react-virtual
- 歌单详情页（可能几百首）
- 搜索结果列表
- 播放历史列表
```
